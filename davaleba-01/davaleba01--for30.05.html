დავალება #1

1. მოედანზე (წითელ კვადრატში მოქცეულ არეალზე) კლიკის დროს უნდა გაიხსნას რამე ვებგვერდი თქვენი გემოვნებით (სპორტთან დაკავშირებული). დახაზული ფოტოს ნაცვლად შეგიძლიათ გამოიყენოთ იგივე ფოტო ამავე ფოლდერიდან, რომელიც არის კვადრატების და წრის გარეშე.

2. ავტობუსზე (მწვანე კვადრატში მოქცეულ არეალზე) კლიკის დროს უნდა გაიხსნას თქვენს მიერ შექმნილი სხვა html ფაილი სახელად ganrigi.html, რომელშიც იქნება დაულაგებელ ლისტში მოქცეული დალაგებული ლისტი. მაგ.:

ამ html ფაილში უნდა იყოს საწყის ფაილზე დასაბრუნებელი ლინკი. ლისთი შეგიძლიათ დაასათაუროთ და გააფორმოთ ბორდერებით და ფერებით. სტილი გაუწერეთ html ფაილშივე (ცალკე css ფაილის შექმნის გარეშე)

3. ტბაზე (ლურჯ წრეში მოქცეულ არეალზე) კლიკის დროს გაიხსნას html ფაილი სახელად tba.html, რომელთანაც დაკავშირებული იქნება tba.css ფაილი სტილებით.

  • ეცადეთ tagname-ების გარდა css ფაილში ასევე გამოიყენოთ კლასები და id.
  • tba.html ფაილში იყოს რამე სათაური, ტბის ფოტო (ფოტოს არ აქვს მნიშვნელობა) და ფოტოს შემდეგ რაიმე შემთხვევითი ტექსტი.
  • ფოტო მოაქციეთ ცენტრში და შემოავლეთ ორმაგი ხაზი.
  • ამ html ფაილშიც უნდა იყოს საწყის ფაილზე დასაბრუნებელი ლინკი.
